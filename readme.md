# 🛠️ Infraestrutura Local - Guia de Comandos rápidos

Este repositório contém o script de automação para o ambiente de desenvolvimento local (Nginx, PHP 8.3, MariaDB, Redis, Supervisor e phpMyAdmin). Abaixo estão os comandos essenciais para gerenciar sua infraestrutura.

---

## 🚀 1. Como Executar o Script de Instalação

Antes de rodar pela primeira vez, dê permissão de execução ao arquivo:

```bash
chmod +x startup.sh
```

Para executar

```bash
bash startup.sh
```
